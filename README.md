# VIM编辑器

   我创建了一个仓储，主要是通过思维导图的方式介绍VIM编辑器的一些用法，仓储总包含思维导图的源文件，以及导出的一张PNG图片，用法均是在linux下的VIM version 7.4.160做了验证，可以保证文中的命令真实有效。基于仓储中的用法，可以更加高效快捷地使用VIM编辑器。

   仓储中包含一个思维导图源文件，思维导图导出的PNG图片，以及README.md和LISENCE.md
